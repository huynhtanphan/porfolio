Data Used:

NashvilleHousing Table: The data file contains property details including unique identifiers, addresses, sale information (date and price), legal references, vacancy status, owner details (names and addresses), land and building characteristics (acreage, tax district, land/building value, total value), and property features (year built, number of bedrooms, full and half bathrooms).

Raw data can be downloaded from: kaggle.com/datasets/tmthyjames/nashville-housing-data and other reputable data sources.
